# 📊 Customer Churn Prediction System (ML + FastAPI + Streamlit)

---

## 📖 Problem Statement
Customer churn is a major challenge for subscription-based businesses.  
The goal of this project is to predict whether a customer is likely to leave the service based on historical data.

Early prediction helps companies:
- Improve customer retention
- Reduce revenue loss
- Target high-risk customers

---

## 🚀 Project Highlights
- 🔹 Machine Learning Model (Random Forest)
- 🔹 FastAPI Backend (Real-time prediction API)
- 🔹 Streamlit Dashboard (Interactive UI)
- 🔹 Probability Score + Visual Charts (Plotly)
- 🔹 End-to-End ML Pipeline

---

## 🛠️ Tech Stack
- Python
- Scikit-learn
- FastAPI
- Streamlit
- Plotly
- Pandas & NumPy

---

## 📂 Dataset Description
The dataset includes:

- Customer demographics (Gender, Senior Citizen, etc.)
- Services subscribed (Internet, Phone, Streaming)
- Account information (Tenure, Contract type)
- Billing details (Monthly & Total Charges)

🎯 Target Variable:
- **Churn (Yes / No)**

---

## 🔍 Approach

1. Data loading using Pandas  
2. Data cleaning and handling missing values  
3. Encoding categorical variables  
4. Feature scaling using StandardScaler  
5. Model training using Random Forest  
6. Saving model using Joblib  
7. Creating API using FastAPI  
8. Building UI dashboard using Streamlit  
9. Integrating API with frontend  

---

## 📊 Model Performance

- ✅ Model Used: **Random Forest Classifier**
- 🎯 Accuracy: ~80% (approx, depends on dataset split)

---

## 📈 Features of Application

- 🔮 Predict customer churn in real-time  
- 📊 Display probability of churn (%)  
- 🟢🔴 Color-based risk indicator  
- 📉 Interactive charts (Gauge + Bar chart)  
- 🧾 User-friendly input form
